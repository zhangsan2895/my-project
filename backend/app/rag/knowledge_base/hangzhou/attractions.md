# 杭州核心景点知识库

city: 杭州
category: attractions
tags: 景点, 旅游, 观光, 打卡

## 西湖
**基本信息**
- 位置与规模：位于浙江省杭州市旧城西侧，面积约6.39平方千米，南北长约3.2千米，东西宽约2.8千米，绕湖一周近15千米。
- 景观格局：形成“一山、二塔、三岛、四堤、五湖”的基本格局。湖中有孤山、小瀛洲、湖心亭、阮公墩等岛屿，苏堤、白堤、杨公堤横贯湖面。
- 荣誉地位：中国大陆首批国家重点风景名胜区、中国十大风景名胜、首批国家5A级旅游景区。2011年列入世界文化遗产名录。
- 开放政策：中国首家且至今少有的不收门票的5A级景区，沿线多处公园（如南线四大公园、太子湾公园等）已实现二十四小时免费开放。

**适合人群**
- 历史文化与自然风光爱好者
- 摄影与写生创作者
- 家庭休闲与日常散步游客

**行程组合建议**
- 四季赏花路线：春季苏堤/白堤赏桃柳与樱花，夏季曲院风荷观荷，秋季满陇桂雨/植物园品桂，冬季灵峰/孤山探梅。
- 经典人文动线：断桥残雪 → 白堤 → 孤山（浙江省博物馆、西泠印社、楼外楼） → 苏堤 → 花港观鱼 → 雷峰塔。

**注意事项**
- 景区为开放式大公园，无统一门禁与门票限制。
- 历史上多次经历疏浚与整治，水质曾受富营养化影响，近年已恢复至III类标准，游览时请注意生态保护。

## 灵隐寺
**基本信息**
- 地理位置：位于杭州西湖西北面，飞来峰与北高峰之间的灵隐山山麓。
- 历史沿革：创建于东晋咸和元年（326年），距今逾1600年。历经十余次毁坏与重建，现存建筑多为清末复建及现代修缮成果。
- 核心建筑与文物：天王殿悬清康熙帝御题“云林禅寺”匾额；大雄宝殿高33.6米，供奉高达19.6米的香樟木释迦牟尼坐像（中国国内最大木雕佛像）；寺内藏有南宋韦陀菩萨遗物、五百罗汉像及云林图书馆。
- 文化关联：相传为南宋高僧济公修行地，与飞来峰造像石窟群相邻。

**适合人群**
- 佛教文化与古建筑研究者
- 祈福与静心参访游客
- 历史遗迹探访者

**行程组合建议**
- 山门参访动线：照壁“咫尺西天” → 理公塔 → 春淙亭 → 飞来峰造像石窟群 → 灵隐寺正门。
- 周边联动：可结合北高峰登山揽胜、法云弄古村落漫步及龙井茶文化体验。

**注意事项**
- 当地导游建议不要拍摄山门外的理公塔，请尊重地方习俗与导览提示。
- 寺院内部部分殿堂（如藏经楼、华严殿）于2002年对外开放，参观时请遵守宗教场所管理规定。

## 西溪国家湿地公园
**基本信息**
- 地理位置与规模：位于杭州市城西，距离西湖约5公里，总面积约11.5平方公里（1150公顷）。
- 景观特色：拥有“秋芦飞雪、火柿映波、深滩会舟、莲滩鹭影”等十景，以及贯穿湿地的福堤、绿堤、寿堤。
- 运营状态：2005年挂牌国家湿地公园，属国家5A级旅游景区。2020年起东区和西区实现一体化管理。
- 交通配套：杭州地铁3号线（西溪湿地南站）与19号线（西溪湿地北站）设站，南侧及北侧均有出入口。

**适合人群**
- 自然生态与湿地科普爱好者
- 亲子家庭与户外徒步游客
- 摄影与自然写生创作者

**行程组合建议**
- 湿地生态体验：乘船游览“深滩会舟”、“曲水寻梅”等水道，沿福堤/绿堤/寿堤进行环湖徒步。
- 文化节点串联：河渚街听曲、高庄宸迹参观、洪园余韵游览，结合当地特产（如羊羔补酒、老三宝、新三宝）体验农家风情。

**注意事项**
- 开放时间分季调整：4月1日至10月31日为8:30-18:30（17:30停售门票）；11月1日至次年3月31日为8:30-17:30（16:30停售门票）。
- 门票政策：全票80元，半票40元（适用6-18周岁青少年及60-69岁长者）；70周岁以上长者、杭州西湖公园年卡持卡人及特定职业/证件持有者可免票入园。
- 票务规则：持有一张门票可在购票当日分别进入东区和西区各一次，周家村出入口具备两区检票功能。

## 雷峰塔
**基本信息**
- 地理位置：位于西湖南岸夕照山雷峰顶上，与北岸宝石山保俶塔隔湖相对。
- 历史渊源：始建于五代吴越国时期（约公元972-977年），原为七层砖木结构佛塔，明代遭战火焚毁仅存砖身，1924年坍塌。2002年在原址按南宋风格重建新塔。
- 建筑特征：新塔总高71.679米，采用钢框架结构，由保护罩、塔身（五层八面）和塔刹组成。底层覆盖古塔遗址，内设液压电梯与钢制楼梯。
- 文化地位：“雷峰夕照”为西湖十景之一；因民间传说《白蛇传》中镇压白娘子的情节而广为人知。

**适合人群**
- 神话传说与文学爱好者
- 历史建筑与考古遗迹考察者
- 登高观景与城市风光拍摄者

**行程组合建议**
- 南北双塔对望：在南山路或长桥公园远眺“一湖映双塔”景观，随后登塔俯瞰西湖全景。
- 文物溯源联动：参观浙江省博物馆孤山馆区“瑞象重明——雷峰塔文物陈列”常设展览，了解地宫出土的鎏金纯银阿育王塔等国宝级文物。
- 公交接驳：可乘坐4路、1314路、西湖内环线等公交车辆抵达龙翔桥、动物园或宋城景区附近。

**注意事项**
- 新塔为文物保护罩与观光塔合一的建筑，登塔需通过电梯或楼梯，注意保管随身物品。
- 塔基遗址与地宫文物具有极高历史价值，参观时请勿触碰保护设施。
- 夕阳西下时段为观赏“雷峰夕照”最佳时机，建议合理安排白天登塔与傍晚观景的时间。
