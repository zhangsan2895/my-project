# 成都行程路线知识库

city: 成都
category: routes
tags: 行程, 路线, 一日游, 二日游, 三日游

## 核心资源索引
- 历史文化地标：武侯祠、杜甫草堂、昭觉寺、华西坝历史建筑群
- 古蜀遗迹集群：金沙遗址、金沙遗址公园、三星堆遗址、鱼凫古城遗址、望丛祠
- 城市生态公园：桂溪生态公园、北湖公园
- 自然地貌单元：龙泉山脉（盆中丘陵）、邛崃山脉、大邑县西岭雪山（大雪塘）、府河、沙河、锦江
- 新兴区域：天府新区、东部新区、高新区

## 一日行程推荐
- 主题：古蜀文明与城市绿肺
- 动线：金沙遗址 → 金沙遗址公园 → 桂溪生态公园 → 北湖公园
- 说明：以成都市区为核心，串联古蜀文化展示空间与现代城市森林公园，快速建立对成都历史脉络与生态建设的认知。

## 二日行程推荐
- 主题：三国底蕴与山水人文
- 动线：
  - D1：武侯祠 → 杜甫草堂 → 府河/沙河沿岸
  - D2：昭觉寺 → 华西坝历史街区 → 龙泉山脉浅丘地带
- 说明：首日聚焦三国政治中心与唐宋文学遗存；次日结合川西禅林文化与抗战时期高校内迁旧址，延伸至东部丘陵地貌。

## 三日行程推荐
- 主题：全域地貌与古蜀溯源
- 动线：
  - D1：市区文化线（武侯祠、杜甫草堂、金沙遗址）
  - D2：平原遗迹线（三星堆遗址、鱼凫古城遗址、望丛祠）
  - D3：山地生态线（邛崃山脉沿线、大邑县西岭雪山大雪塘）
- 说明：由中心城区向西北及西南延伸，覆盖成都平原腹地至横断山脉东缘，完整呈现“平原-丘陵-高山”三级阶梯地貌。

## 分群体变体建议
- 亲子家庭：优先安排金沙遗址公园、桂溪生态公园、北湖公园等开阔绿地；结合平原农业与水利背景开展自然观察。
- 银发长者：侧重低强度平地游览，推荐武侯祠、杜甫草堂、府河/锦江沿岸步道；避开邛崃山脉等高海拔区域，利用秋季漫长、冬季温暖的气候特征出行。
- 摄影爱好者：捕捉龙泉山脉植被与浅丘起伏轮廓；拍摄邛崃山脉雪峰（大雪塘）与云海；记录府河、沙河穿城水系与古建（武侯祠、昭觉寺）光影。
- 低强度休闲：以中心城区公园网络为主（桂溪生态公园、北湖公园、金沙遗址公园），沿平原地带缓行，减少体力消耗。

## 天气应对方案
- 雨季（夏季降水充沛，月均降水量超100毫米）：转为室内或林荫动线，参观金沙遗址展馆、武侯祠、杜甫草堂；利用桂溪生态公园、北湖公园等大型森林公园树冠遮蔽避雨。
- 高温期（7月偶现极端高温，历史最高温可达37.5℃）：调整至清晨或傍晚活动；优先选择河网密集区（府河、沙河、锦江沿岸）与建成区绿化覆盖率高的公园地带；避免正午前往龙泉山脉或邛崃山脉等高差较大区域。

## 交通与动线提示
- 地形适配：全市以平原为主（占比40.1%），地势西北高东南低，整体呈扇形水系分布，城区道路与河网走向一致，利于平面动线规划。
- 水系利用：锦江、府河、沙河贯穿主城区，可沿河道绿道进行短途接驳与慢行游览。
